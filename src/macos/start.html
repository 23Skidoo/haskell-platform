What is Installed

On Mac OS X, the Haskell Platform is installed in two major pieces: GHC and Haskell Platform. They are installed respectively in:

/Library/Frameworks/GHC.framework
/Library/Haskell

Executables are symlinked in /usr/bin and should be available in any shell.

Installed Haskell Package Layout

To facilitate easier management, the non-core libraries of Haskell Platform are installed under a compiler-specific subdirectory of /Library/Haskell, each in their own per-package directory.

To configure cabal to install additional packages with the same layout, set your .cabal/config file to (replacing xxx with your user name):

library-profiling: True
documentation: True

install-dirs global
  prefix: /Library/Haskell/$compiler/lib/$pkgid
  libsubdir:  
  datasubdir: 
  docdir: $prefix/doc
  
install-dirs user
  prefix: /Users/xxx/Library/Haskell/$compiler/lib/$pkgid
  libsubdir:  
  datasubdir: 
  docdir: $prefix/doc
