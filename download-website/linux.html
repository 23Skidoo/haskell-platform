Build from source

For Unix systems (including Mac OS X), there is a source installer.

haskell-platform-2011.2.0.0.tar.gz

You need GHC 7.0.2 installed before building the platform. You can get this from your distro or you can get a GHC 7.0.2 generic binary.

To install from source download and unpack the installer, then (possibly with 'sudo'):

    ./configure
    make
    make install

You may pass --prefix flags to change the default install path.
