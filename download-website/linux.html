Build from source

For Unix systems (including Mac OS X), there is a source installer.

haskell-platform-2010.1.0.0.tar.gz

You need GHC 6.12.3 installed before building the platform. You can get this from your distro or you can get a GHC 6.12.3 generic binary.

To install from source download and unpack the installer, then (possibly with 'sudo'):

    ./configure
    make
    make install

You may pass --prefix flags to change the default install path.

Build from cabal

If you already have a reasonable Haskell development environment with GHC 6.12 and cabal-install, you can build the platform from the Cabal package alone.

Haskell Platform Cabal Package

To install, unpack the cabal tarball, and run:

    cabal install
