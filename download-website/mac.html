MacPorts

The Haskell Platform is also in MacPorts. If you have MacPorts installed, you can build the Haskell Platform by typing:

     sudo port install haskell-platform

Build from Source

If you already have a GHC 6.12, but not the full platform, you can build it from source on the Mac:

haskell-platform-2010.1.0.0.tar.gz

You need GHC 6.12.1 installed before building the platform.

To install from source download and unpack the installer, then (possibly with 'sudo'):

    ./configure
    make
    make install

You may pass --prefix flags to change the default install path.
