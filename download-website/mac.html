Download Haskell Platform 2011.4.0.0 32 bit (10.6 or later)

Download Haskell Platform 2011.4.0.0 64 bit (10.6 or later)

After downloading:

  • Double click the package icon to start the installer.
  • Follow the instructions.

Notes:

  • Only one architecture (32-bit or 64-bit) can be installed at a time.

  • Xcode is required prior to installation. Xcode can be downloaded from Apple's developer website. Use either Xcode 3.2 or later, or Xcode 4 or later.

For Mac OS X 10.5 please use the 2010.2.0.0 installer, below.
