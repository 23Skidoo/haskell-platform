Haskell Platform for OSX

    For Mac OS X Leopard (10.5), Snow Leopard and above (including 64 bit):

    • Download Haskell for Mac OS X

    After downloading:

    • Open the .dmg file
    • Follow the install instructions

    MacPorts

    The Haskell Platform is also in MacPorts. If you have MacPorts installed, you can build the Haskell Platform by typing:

         sudo port install haskell-platform
